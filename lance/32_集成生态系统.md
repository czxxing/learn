# 第三十二章：集成生态系统

## 🎯 核心概览

Lance 与主流数据处理工具无缝协作，包括 Pandas、Polars、DuckDB 等，通过标准格式（Apache Arrow）实现零拷贝数据交换。

## 🐼 Pandas 集成

### 读写 Pandas DataFrame

```python
import pandas as pd
import lance
import numpy as np

# 创建 Pandas DataFrame
df = pd.DataFrame({
    "id": range(1000),
    "embedding": [np.random.randn(768).astype(np.float32) for _ in range(1000)],
    "text": [f"doc_{i}" for i in range(1000)],
    "score": np.random.uniform(0, 100, 1000),
})

# 写入 Lance（自动转换为 Arrow）
table = lance.write_table(df, uri="data.lance")

# 读取为 Pandas DataFrame
df_back = table.to_pandas()
print(df_back.head())

# 零拷贝转换（仅限数值列）
arrow_table = table.to_arrow()
df_arrow = arrow_table.to_pandas()
```

### 增量更新

```python
# 追加数据
new_df = pd.DataFrame({
    "id": [1000, 1001],
    "embedding": [np.random.randn(768).astype(np.float32) for _ in range(2)],
    "text": ["doc_1000", "doc_1001"],
    "score": [50.0, 75.0],
})

table.add(new_df)  # 追加新行

# 更新特定行
update_df = pd.DataFrame({
    "score": [99.0],
})
table.update(update_df, where="id == 1000")

# 删除行
table.delete("id > 500")
```

## 📊 Polars 集成

### Polars 互操作

```python
import polars as pl
import lance

# 创建 Polars DataFrame
df_polars = pl.DataFrame({
    "id": range(1000),
    "embedding": [list(np.random.randn(768)) for _ in range(1000)],
    "text": [f"doc_{i}" for i in range(1000)],
})

# 转换为 Lance（通过 Arrow）
table = lance.write_table(df_polars, uri="data.lance")

# 从 Lance 读取为 Polars
df_back = pl.from_arrow(table.to_arrow())

# 使用 Polars 的快速查询语法
results = (
    df_back
    .filter(pl.col("id") > 100)
    .select(["id", "text"])
)
```

## 🦆 DuckDB 集成

### SQL 查询

```python
import duckdb
import lance

# 注册 Lance 表为虚拟表
table = lance.open("data.lance")
duckdb.register_arrow("docs", table.to_arrow())

# 使用 DuckDB SQL 查询
result = duckdb.sql("""
    SELECT id, text, score
    FROM docs
    WHERE score > 50
    ORDER BY score DESC
    LIMIT 10
""").arrow()

print(result)
```

### OLAP 查询

```python
# 复杂的分析查询
result = duckdb.sql("""
    SELECT 
        COUNT(*) as count,
        AVG(score) as avg_score,
        STDDEV(score) as std_dev
    FROM docs
    GROUP BY CAST(id / 100 AS INT)
    HAVING count > 5
""")

result.show()
```

## 📍 Apache Arrow 兼容性

### 零拷贝转换

```rust
use arrow::record_batch::RecordBatch;
use arrow::array::*;

pub fn lance_to_arrow(batch: &LanceRecordBatch) -> RecordBatch {
    // Lance 内部使用 Arrow 数组
    // 直接转换无需复制
    batch.to_arrow_record_batch()
}

pub fn arrow_to_lance(batch: &RecordBatch) -> LanceRecordBatch {
    // 反向转换也是零拷贝
    LanceRecordBatch::from_arrow(batch)
}
```

## 🔗 其他生态集成

### 与 Spark 的集成

```python
from pyspark.sql import SparkSession
import lance

spark = SparkSession.builder.appName("LanceExample").getOrCreate()

# 读取 Lance 表为 Spark DataFrame
lance_table = lance.open("data.lance")
df_spark = spark.createDataFrame(
    lance_table.to_arrow().to_pandas()
)

# Spark 处理后写回 Lance
processed = df_spark.filter("score > 50")
lance.write_table(processed.toPandas(), uri="processed.lance")
```

### 与 Dask 的分布式处理

```python
import dask.dataframe as dd
import lance

# 从 Lance 读取大数据集
table = lance.open("large_data.lance")
ddf = dd.from_delayed(
    [table.to_pandas()],
    meta=table.schema()
)

# 分布式处理
result = ddf[ddf["score"] > 50].compute()

# 写回
lance.write_table(result, uri="processed.lance")
```

## 📊 性能对比

```python
import time
import numpy as np
import pandas as pd
import lance

# 数据规模：100 万行
data = {
    "id": range(1000000),
    "embedding": [np.random.randn(768).astype(np.float32) for _ in range(1000000)],
    "score": np.random.uniform(0, 100, 1000000),
}

# 方案1：Pandas in-memory
start = time.time()
df = pd.DataFrame(data)
result = df[df["score"] > 50]
print(f"Pandas: {time.time() - start:.3f}s")

# 方案2：Lance 向量搜索
table = lance.write_table(data, uri="data.lance")
start = time.time()
results = table.search(data["embedding"][0]).limit(10).to_list()
print(f"Lance: {time.time() - start:.3f}s")

# 方案3：DuckDB
duckdb.register_arrow("data", table.to_arrow())
start = time.time()
result = duckdb.sql("SELECT * FROM data WHERE score > 50").arrow()
print(f"DuckDB: {time.time() - start:.3f}s")
```

## 📚 总结

Lance 的生态集成提供：

1. **Pandas/Polars**：DataFrame 互操作
2. **DuckDB**：高效 SQL 查询
3. **Arrow**：零拷贝数据交换
4. **Spark/Dask**：分布式处理
5. **标准格式**：与任何 Arrow 兼容工具协作

这使得 Lance 可以无缝集成到现有的数据处理流水线中。
