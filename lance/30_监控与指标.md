## 📊 性能指标收集

### 核心指标

```rust
pub struct PerformanceMetrics {
    // 延迟相关
    pub query_latency_p50: Duration,    // 中位数
    pub query_latency_p95: Duration,    // 95 分位
    pub query_latency_p99: Duration,    // 99 分位
    
    // 吞吐量
    pub queries_per_second: f64,
    pub rows_per_second: f64,
    
    // 缓存
    pub cache_hit_rate: f32,
    pub cache_miss_rate: f32,
    
    // 索引
    pub index_scan_count: u64,
    pub index_cache_hits: u64,
    
    // 错误
    pub error_count: u64,
    pub error_rate: f32,
}

pub struct MetricsCollector {
    histogram: Arc<Histogram>,
    counter: Arc<Counter>,
}

impl MetricsCollector {
    pub fn record_query(&self, duration: Duration) {
        self.histogram.observe(duration.as_secs_f64());
    }
    
    pub fn record_cache_hit(&self) {
        self.counter.inc_with_labels(&["type", "cache_hit"]);
    }
}
```

## 🔍 Tracing 与分布式追踪

### 追踪跨度

```rust
use tracing::{span, event, Level};

pub async fn search_with_tracing(
    query: &[f32],
    k: usize,
) -> Result<Vec<SearchResult>> {
    let span = span!(Level::DEBUG, "vector_search", k = k);
    let _guard = span.enter();
    
    // Step 1: 向量预处理
    let prep_span = span!(Level::DEBUG, "preprocess");
    let _prep = prep_span.enter();
    let normalized = normalize_vector(query)?;
    drop(_prep);
    event!(Level::DEBUG, "preprocessing complete");
    
    // Step 2: 索引查询
    let index_span = span!(Level::DEBUG, "index_search");
    let _index = index_span.enter();
    let candidates = search_index(&normalized, k * 4).await?;
    drop(_index);
    
    // Step 3: 重排
    let rerank_span = span!(Level::DEBUG, "reranking");
    let _rerank = rerank_span.enter();
    let results = rerank_results(candidates, query, k).await?;
    drop(_rerank);
    
    Ok(results)
}
```

## 📈 告警与日志

```rust
pub struct AlertManager {
    thresholds: AlertThresholds,
}

pub struct AlertThresholds {
    pub max_query_latency: Duration,
    pub min_qps: f64,
    pub max_error_rate: f32,
}

impl AlertManager {
    pub fn check(&self, metrics: &PerformanceMetrics) -> Vec<Alert> {
        let mut alerts = Vec::new();
        
        if metrics.query_latency_p99 > self.thresholds.max_query_latency {
            alerts.push(Alert {
                level: AlertLevel::Warning,
                message: format!(
                    "High latency: p99 = {:?}",
                    metrics.query_latency_p99
                ),
            });
        }
        
        if metrics.error_rate > self.thresholds.max_error_rate {
            alerts.push(Alert {
                level: AlertLevel::Critical,
                message: format!(
                    "High error rate: {:.2}%",
                    metrics.error_rate * 100.0
                ),
            });
        }
        
        alerts
    }
}
```

## 📚 总结

完整的监控体系包括：
1. **性能指标**：延迟、吞吐量、缓存命中率
2. **分布式追踪**：跟踪请求生命周期
3. **调试工具**：分析执行计划、火焰图
4. **告警系统**：实时发现异常

这些工具帮助开发者快速定位性能问题。
