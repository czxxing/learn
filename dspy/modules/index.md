# DSPy 模块详解

> 本目录包含 DSPy 核心模块的详细分析文档。

## 目录

### 核心模块

| 模块 | 文档 | 描述 |
|------|------|------|
| Module | [module.md](module.md) | 所有 DSPy 模块的基类，提供统一调用入口 |
| Predict | [predict.md](predict.md) | 基础预测模块，封装 LM 调用 |
| Adapter | [adapter.md) | 格式转换层，适配不同 LM 接口 |
| LM | [lm.md](lm.md) | 语言模型统一封装层 |
| Signature | [signature.md](signature.md) | 输入输出规范定义 |
| Prediction | [prediction.md](prediction.md) | 执行结果封装 |
| Teleprompter | [teleprompter.md](teleprompter.md) | 自动优化器 |
